## What is BonusRound?

BonusRound is a Minecraft server that brings dodgeball to Minecraft. The server features a skill-based matchmaking system,
leaderboards, and abilities such as double jumping and dashing to enhance gameplay. Players can join matches, earn ranked
rating, and climb the leaderboards to become the top dodgeball player.

### Features

- Skill-based matchmaking system
- Dodgeball physics/hit detection
- Abilities: double jump, dash
- Leaderboards
- Currencies and cosmetics \*(work in progress)\*\*
