## The History

This website was built after several iterations. The first version was a less dynamic site built with mostly shadcn
components. After several attempts to design a new portfolio, I settled on the current design which focuses on user
experience and design.

### Inspiration

- [https://jasoncameron.dev/](https://jasoncameron.dev/)
- [https://floriankiem.com/](https://floriankiem.com/)
- [https://tr.af/](https://tr.af/)
- [https://ayaneshu.com/](https://ayaneshu.com/)
- Several other projects on [Pafolios](https://pafolios.com/) and [Awwwards](https://www.awwwards.com/)

### Features

- A cool navbar that doubles as a breadcrumb
- Project cards and overviews powered by MDX
- Dark mode as every modern website should have
- A blog powered by Supabase, MDX, and Github auth
- Micro animations

### Technologies

- [Next.js](https://nextjs.org/)
- [Motion](https://motion.dev/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Supabase](https://supabase.com/)
